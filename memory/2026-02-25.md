## API 端点

### 认证
- `POST /admin/auth/signup` - 注册
- `POST /admin/auth/signin` - 登录
- `POST /admin/auth/forgot-password` - 忘记密码

### 用户
- `GET /admin/quota/me` - 获取余额
- `GET /admin/quota/transactions` - 交易记录
- `POST /admin/quota/recharge` - 充值
- `GET /admin/packages` - 套餐列表
- `POST /admin/packages/purchase` - 购买套餐

### 支付
- `POST /admin/stripe/checkout` - Stripe 支付
- `POST /api/payment/epusdt/callback` - EPUSDT 回调
- `POST /api/payment/stripe/webhook` - Stripe 回调

### 运营统计
- `GET /admin/stats/overview` - 总览数据
- `GET /admin/stats/users/trend` - 用户趋势
- `GET /admin/stats/revenue/trend` - 收入趋势
